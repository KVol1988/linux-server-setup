# Linux Server Automated Setup

Scripts for automatic deployment and configuration of Linux servers.

## Features
- Automatic software installation
- Security hardening
- User management
- Service configuration

## Usage
```bash
# Clone repository
git clone https://github.com/yourname/linux-server-setup

# Run setup script
chmod +x setup.sh
./setup.sh
```

## Technologies
- Ubuntu/CentOS
- Bash scripting
- SSH
- Apache/Nginx
